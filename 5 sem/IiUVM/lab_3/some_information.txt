Теория:
 
  HDD (hard disk drive, жесткий диск) – это запоминающее устройство, основанное на принципе магнитной записи.
  
  SSD-накопитель — это носитель данных, в котором, в отличие от жесткого диска, используется энергонезависимая (флеш) память для хранения данных и доступа к ним.
  
  Магнитная запись информации основана на том, что многие материалы в магнитном поле намагничиваются вдоль его линий и сохраняют эту намагниченность даже после отключения поля.
  
  Основные компоненты жестких дисков:
- диски;
- головки чтения/записи;
- механизм привода головок;
- двигатель привода дисков;
- печатная плата со схемами управления;
- кабели и разъемы;
- элементы конфигурации (перемычки и переключатели).

  Существует несколько типов магнитной записи информации на жесткий диск:
  1. Longitudinal recording (продольная запись) - В этом методе магнитные частицы выравниваются вдоль направления записи и создаются магнитные области, которые представляют собой биты информации.
  2. Perpendicular recording (перпендикулярная запись) - В этом методе магнитные частицы выравниваются перпендикулярно направлению записи, что позволяет увеличить плотность записи и увеличить емкость жесткого диска.
  3. Shingled magnetic recording (шингловая магнитная запись) - В этом методе магнитные дорожки перекрываются друг на друга, похоже на крыши шинглов, что позволяет увеличить плотность записи.
  4. Heat-assisted magnetic recording (HARM) (магнитная запись с помощью тепла) - это метод записи, который использует лазер для нагрева магнитной пленки перед записью. Это позволяет увеличить плотность записи и емкость жесткого диска.

  Устройства внешней памяти могут иметь различные характеристики, включая емкость, скорость передачи данных, интерфейс подключения и форм-фактор.
  
  1. Емкость - это объем данных, который может хранить устройство внешней памяти.

  2. Скорость передачи данных - это скорость, с которой устройство внешней памяти может передавать данные на компьютер или другое устройство.

  3. Интерфейс подключения - это тип подключения между устройством внешней памяти и компьютером или другим устройством.

  4. Форм-фактор - это размер и форма устройства внешней памяти.

  Сервоячейка - это механизм, который используется для управления положением или движением объекта с помощью вращения вала.

  Сервометка - это устройство, которое используется для измерения угла поворота или скорости вращения.

  Серводорожка - это система управления, которая используется для контроля движения поездов или других транспортных средств на железнодорожных путях.

  Внешняя память - это устройство, которое используется для хранения данных на компьютере или другом электронном устройстве.

  1. Жесткий диск (HDD) - это основной носитель информации на компьютере.

  2. Твердотельный накопитель (SSD) - это более быстрый и надежный аналог жесткого диска.

  3. Оптический диск (CD/DVD/Blu-ray) - это устройство для чтения и записи оптических дисков.

  4. Флеш-накопитель - это маленькое устройство, которое используется для хранения и передачи файлов между компьютерами.

  5. Облачное хранилище - это сервис, который позволяет хранить данные на удаленных серверах через интернет.

  По исполнению:

  1. Портативные - маленькие устройства, которые можно легко переносить с собой.

  2. Настольные - устройства, которые обычно используются на рабочем столе и не предназначены для переноски.

  По конструкции:

  1. Внешние - устройства, которые подключаются к компьютеру через интерфейс USB, FireWire или другой.

  2. Внутренние - устройства, которые устанавливаются внутрь компьютера и подключаются к материнской плате.

  По принципу адресации и доступа:

  1. Случайный доступ (RAM) - устройства, которые позволяют быстрый доступ к данным в случайном порядке.

  2. Последовательный доступ (SD) - устройства, которые позволяют доступ к данным только в определенном порядке.

  По типу использования физического явления:

  1. Магнитные - устройства, которые используют магнитное поле для записи и чтения данных.
  
  2. Оптические - устройства, которые используют лазер для записи и чтения данных.

  3. Электронные - устройства, которые используют электрические сигналы для записи и чтения данных.
  
  Флеш-накопители - устройства, которые используют флэш-память для хранения данных и подключаются к компьютеру через интерфейс USB.

  Жесткие диски (HDD) - устройства, которые используют магнитное поле для хранения данных и подключаются к компьютеру через интерфейс USB, FireWire или eSATA.

  SSD-накопители - устройства, которые используют электронную память для хранения данных и обеспечивают более быстрый доступ к данным, чем HDD.
  
  Карты памяти - устройства, которые используются в цифровых камерах, мобильных устройствах и других устройствах для хранения данных и подключаются к компьютеру через картридер.

  Оптические диски - устройства, которые используют лазер для записи и чтения данных и включают CD, DVD и Blu-ray диски.

  1. Кодирование Хаффмана - метод сжатия данных, который основан на использовании битовых последовательностей разной длины для кодирования символов.

  2. Кодирование Лемпеля-Зива - метод сжатия данных, который основан на построении словаря из уже встреченных последовательностей символов и замене этих последовательностей на ссылки на словарь.

  3. Кодирование RLE (Run Length Encoding) - метод сжатия данных, который основан на замене повторяющихся символов на количество повторений и сам символ.

  4. Кодирование арифметическое - метод сжатия данных, который основан на использовании вероятностей символов для определения диапазона возможных значений и кодирования
символов в этом диапазоне.

  5. Кодирование Дельта - метод сжатия данных, который основан на замене значений на разницу между ними и предыдущим значением.
